What?

This project set out to try to make a menu style that roughly mimiced binder dividers.i

This could be applied in two ways:

- It was originally concieved as a way to narrow down a search. One tab could have a location icon, one could have a price icon, etc

- It ended up being clean and simple enough to actually be used as a nav bar

How?

This demo is almost entirely CSS. In fact, I just hav a div surrounding each section of text - a total of 4 - and the rest is done in css. It uses some fancy borders for the trapezoids, some pseudo elements for the tabs, and the beloeved css transitions for the animation. No javascript require (except to set the height, but thats just a little addition of user expirience #ocd)

Why?

This project was part of a (design-oriented) hack-a-day projects. Each day I decide to make something - sometimes I'll base it off of something else on the web, but I always make sure to put my own unique spin on it. This project is the 4th (0 indexed) in the series.

Although it's terrible practice, I make all of the application one-page. In the scary way. Just one html file. No dependancies, style in the header javascript in the body - pretty much the oppisite of what you find anywhere on the web. It's terrible practice but great fun, and more than educational.
